Technologies

Javascript is an interpreted programming language, dialect of the ECMAScript standard. It is defined as object-oriented, 3 based on prototypes, imperative, weakly typed and dynamic.

Angular is a framework for web applications developed in TypeScript, open source, maintained by Google, which is used to create and maintain web applications on a single page.

Node.js is a cross-platform, open-source runtime environment for the server layer (but not limited to it) based on the ECMAScript programming language, asynchronous, with I / O data in an event-oriented architecture and based on the Google V8 engine.

Git is a version control software designed by Linus Torvalds, thinking about the efficiency and reliability of maintaining application versions when they have a large number of source files. Its purpose is to keep track of changes in computer files and coordinate the work that several people perform on shared files.

PostgreSQL, also known as Postgres, is a free and open source relational database management system (RDBMS) that emphasizes the extensibility and compliance with technical standards. It is designed to handle a variety of workloads, from individual machines to data stores or web services with many concurrent users. It is the default database for macOS Server

Web Resposive is a design and development philosophy whose objective is to adapt the appearance of web pages to the device that is being used to visit them.

HTML refers to the markup language for the development of web pages. It is a standard that serves as a reference for software that connects with the development of web pages in its different versions, defines a basic structure and a code (called HTML code) for the definition of content of a web page, such as text, images, videos , games, among others.

CSS is a graphic design language to define and create the presentation of a structured document written in a markup language. It is widely used to establish the visual design of web documents, and user interfaces written in HTML or XHTML.
